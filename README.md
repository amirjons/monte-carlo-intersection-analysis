# Анализ метода Монте-Карло для оценки площади пересечения кругов

## Описание

Этот репозиторий содержит решение задачи A1, включающее:
- Реализацию алгоритма Монте-Карло для задачи A1i
- Экспериментальный анализ точности метода
- Визуализацию результатов и содержательные выводы

## Структура
- src/ # Исходные коды
-  data/ # Данные эксперимента
-   esults/ # Графики и визуализации
-   docs/ # Отчеты и документация
-   README.md

## Для работы кода на python надо установить библиотеки:
- pip install pandas
- pip install matplotlib
