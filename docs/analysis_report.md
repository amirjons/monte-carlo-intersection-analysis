# Отчет по задаче A1: Анализ метода Монте-Карло

## Точное значение площади

Точная площадь пересечения трех кругов вычисляется по формуле:

\[ S = 0.25 \cdot \pi + 1.25 \cdot \arcsin(0.8) - 1 \]

**Точное значение:** \( S_{exact} = 0.944517 \)

## Методика эксперимента

### Параметры эксперимента:
- **Диапазон N:** от 100 до 100 000 точек с шагом 500
- **Области генерации:**
  - Широкая: [0,3]×[0,3] (площадь = 9)
  - Узкая: [1,2]×[1,2] (площадь = 1)
- **Количество повторений:** 10 для каждого N (усреднение)

## Ключевые результаты

### 1. Эффективность различных областей генерации

**Узкая область** значительно превосходит **широкую область** по точности:

- При N = 100 000:
  - Узкая область: ошибка 0.15%
  - Широкая область: ошибка 1.32%

### 2. Скорость сходимости

- **Узкая область** достигает точности 1% при N ≈ 1 500 точек
- **Широкая область** требует N ≈ 25 000 точек для той же точности

### 3. Стабильность оценок

Оценки по **узкой области** демонстрируют:
- Меньший разброс значений
- Более плавную сходимость
- Предсказуемое поведение ошибки

## Выводы

1. **Выбор области генерации** критически важен для эффективности метода Монте-Карло
2. **Узкая область** обеспечивает ускорение сходимости в ~16 раз по сравнению с широкой областью
3. **Рекомендация:** всегда использовать минимальную ограничивающую область, содержащую целевую фигуру
4. **Для практических применений** метода Монте-Карло предварительный анализ геометрии задачи обязателен

## Объяснение результатов

Разница в эффективности объясняется соотношением площадей:
- Узкая область: целевая фигура занимает ~94% площади
- Широкая область: целевая фигура занимает ~10% площади

Это приводит к разной "плотности" полезных бросков и, соответственно, разной скорости сходимости.
